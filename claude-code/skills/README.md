# Skills

Claude Code skills are Markdown files that define reusable slash commands.

## Format

Each skill is a `.md` file. The filename becomes the command name (e.g. `review.md` → `/review`).

## Installation

```bash
# Global (available in all projects)
cp *.md ~/.claude/skills/

# Project-local
cp *.md /your-project/.claude/skills/
```
