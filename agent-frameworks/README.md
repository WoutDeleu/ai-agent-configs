# Agent Frameworks

Configuration templates for LLM agent orchestration frameworks.

## Contents

| Directory | Framework |
|-----------|-----------|
| `langchain/` | LangChain / LangChain Expression Language |
| `crewai/` | CrewAI multi-agent framework |
| `langgraph/` | LangGraph stateful agent graphs |

Each subdirectory contains config templates and usage notes specific to that framework.
